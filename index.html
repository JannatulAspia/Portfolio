Research Experience

Detailed overview of my research projects

Kidney Stone Classification (Ensemble-ASNet)

Developed a weighted ensemble of InceptionV3, VGG16, and MobileNetV2 for accurate and interpretable kidney stone detection from CT scans. Integrated Grad-CAM++ for model interpretability and optimized ensemble weights for combining class probabilities.

Achievements: 99.81% accuracy, precision, recall, and F1-score.

Deep Learning Medical Imaging Grad-CAM++

Pavement Crack Detection using YOLOv9

Built an automated system for real-time detection of four pavement crack types (pothole, longitudinal, transverse, alligator) using a custom-annotated dataset of 1,200 images.

Achievements: 73.2% F1-score and 70.4% mAP.

YOLOv9 Object Detection Infrastructure

Smoker Status Prediction using Machine Learning

Designed predictive models to classify smoking status of individuals in Bangladesh using health and demographic features (age, height, weight, waist circumference, vision, hearing, blood pressure).

Achievements: HEStacked model achieved 88.03% precision.

ML Algorithms Healthcare Public Health
