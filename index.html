Research Experience

Kidney Stone Classification (Ensemble-ASNet)

Under Review as: ‚ÄúEnsemble-ASNet: Weighted Transfer Learning Framework with Grad-CAM++ for Robust Kidney Stone Classification‚Äù (IEEE EICT, 2025)

Developed a weighted ensemble of InceptionV3, VGG16, and MobileNetV2 for accurate and interpretable kidney stone detection from CT scans. Integrated Grad-CAM++ for model interpretability and optimized ensemble weights for combining class probabilities. Achieved 99.81% accuracy, precision, recall, and F1-score, validating clinical relevance and highlighting important regions in scans. Demonstrated a robust, precise, and explainable approach with potential for clinical deployment and multi-institutional dataset expansion.

Pavement Crack Detection using YOLOv9

Under Review as: ‚ÄúReal-Time Pavement Crack Detection Using YOLOv9: A Deep Learning Approach for Infrastructure Monitoring‚Äù (IEEE EICT, 2025)

Built an automated system for real-time detection of four pavement crack types (pothole, longitudinal, transverse, alligator) using a custom-annotated dataset of 1,200 images. Trained and evaluated YOLOv9, achieving 73.2% F1-score and 70.4% mAP. The system provides precise, scalable, and rapid crack detection, supporting informed decision-making for road maintenance and infrastructure safety.
